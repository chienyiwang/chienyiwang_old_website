如何快速下載Coursera課程資料？

發表於 2016-06-21   |   分類於 Technology   |     |   次瀏覽   |  

MOOC 越來越盈利化

Coursera也算是線上教育MOOC(Massive Open Online Course)的始祖了，2012年一推出的時候，課程其實沒有太多選擇。不過幾個Computer Science的課程都是來自美國有名的大學，內容也做得還滿好的，想當初開始對於一些CS的課程產生興趣，常常會去Coursera或是Udacity找好玩的課程來註冊。這樣的課程通常一集影片大概只有5-10分鐘，設計方式還滿不錯的，能夠在短暫的時間內完成一些小作業並得到些成就感。但是大量註冊兩三個以上的課程之後，其實根本無法與學校的課程一起兼顧，在沒有強大誘因以及自制力的情況下，往往是先捨棄了線上課程，想說之後再回來看影片就好了，想當然很少有真的再回來看…很高興看到台大很多課程也都陸續放在上面，包括葉丙成老師的機率課，還有林軒田老師的機器學習。很感謝林軒田老師對於線上課程的用心，讓我能夠在替代役的時期，順利修到了大學後悔沒有修到的課程。

直到去年大概又看到一些大家推薦的課程，想要回來看看，才發現整個網站風格都不一樣了。大概是因為獲得了VC的融資，因此現在的首頁上充滿了各種套裝課程的促銷(Nano-degree)，也登出了許多找到工作的人使用後的心得，想要說服大家購買套裝課程。當初可以免費觀看的課程，現在想要找到免費觀看的連結都有些困難（有興趣尋找免費連結的可以參考這篇知乎：Coursera 是开始全面收费了么？)

轉移新平台後 舊平台某些課程即將消失

目前有報導指出，Coursera將於6/30捨棄舊平台並整合大部分的課程到新平台。Coursera官方發表說明維護兩個平台需要顯著的資源，為了進一步改善新平台的體驗，Coursera決定停止舊平台的服務。但有人就發現了其實滿多熱門的課程都不會繼續在新平台上供應，因此開始去尋找可以快速下載這些課程的方法。我自己也參考了幾個資源，試過了幾個方法，以下簡單整理一下使用的心得：

如何快速尋找會消失的課程

Class Central整合了超過450堂課程將跟隨著舊平台一起消失的列表。可以先從裡面看到已經分類好的課程，以及大家對課程的評價，來決定要下載哪些課程。

如何下載課程的所有資料？(新平台上的課程也可下載)

方法一：直接暴力下載

如下圖所示，最直接的方法是進入課程，點擊課程的Lecture或是Video Lecture頁面，可以在右邊發現講義、影片以及字幕下載的連結：

但這樣的方法太慢了，於是有人開發了Plug-in還有一些軟體包來幫助批次下載。

方法二：安裝Google Chrome Plug-in “Course Materials Downloader” (限已註冊課程)

  1. 下載Plug-in Course Materials Downloader。

  2. 進入已經註冊的課程頁面，點選Lecture的分頁。

  3. 點選Plug-in，會新開一個分頁，進入下載的頁面。在此可以做設定，選擇要下載哪些檔案。

  4. 點選藍色的”Start Download”按鈕，就會自動開始下載了。但這樣的方法缺點是全部的檔案都會下載到同一個位置，沒有分門別類好。如果想要自動分類好不同的資料夾，可以使用方法三。

方法三：使用Python package “coursera-dl”

詳細的使用方法可以參考作者的github連結[2]，這邊說明最直接的做法。

  1. (Recommended for all Operating Systems) 用pip下載package
    pip install coursera-dl

  2. 在課程頁面網址，我們可以找到課程的id。如：/class.coursera.org/mmds-001/的id就是”mmds-001”。若是覺得要點進去課程頁面看網址很麻煩，在Class Central裡的課程介紹，若是已經要停止供應的課程， 一開始會有個紅色框框，警告此課程在6/30之後就無法使用，下面可以找到課程的id：

    有些id可能是無法被下載的，通常選比較前面一點的比較不會失敗。

  3. 有了course id之後，輸入下面指令就會開始下載了。
    coursera-dl -u <Coursera帳號/信箱> -p <密碼> <id_1> <id_2> ... <id_n>
    Ex: coursera-dl -u dora12402@gmail.com -p Password1 mmds-001 nlp
